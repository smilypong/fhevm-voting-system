# 🚀 Quick Start Guide

## 📋 Prerequisites
- Node.js (v16 or later)
- MetaMask wallet
- Git

## ⚡ 5-Minute Setup

### 1. Clone & Install
```bash
git clone <your-repo-url>
cd fhevm-new-project
npm install
```

### 2. Compile & Test
```bash
npm run compile
npm run test
```

### 3. Run Demo
```bash
npm run demo
```

### 4. Deploy Contract
```bash
npm run deploy-simple
```

### 5. Open Frontend
```bash
# Open in browser
open frontend/voting-app.html
```

## 🔗 Important Links
